# Contributing to Foosball

Thank you for your interest in contributing to this project! We welcome pull requests and suggestions from the community.

## How to Contribute

1. **Fork the repository**
   - Click the "Fork" button at the top right of this page to create your own copy of the repository.
2. **Clone your fork**
   - Run `git clone https://github.com/<your-username>/foosball.git` and then `cd foosball`.
3. **Create a branch**
   - Create a new branch for your change: `git checkout -b feature/your-feature-name`
4. **Make your changes**
   - Commit your changes with clear messages: `git commit -m "Describe your change"`
5. **Push to your fork**
   - `git push origin feature/your-feature-name`
6. **Open a Pull Request**
   - Go to your fork on GitHub and click "Compare & Pull Request". Please describe your changes clearly.

## Review Process
- All PRs require review before merging.
- Please address any feedback and update your PR as needed.
- Once approved, your PR will be merged.

## Code of Conduct
Please be respectful and follow the [GitHub Community Guidelines](https://docs.github.com/en/github/site-policy/github-community-guidelines).

## Questions or Issues
If you have any questions, feel free to open an issue or start a discussion.
